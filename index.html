Proyecto Glowline 2025

Idea del Proyecto

Implementación de un sistema inteligente de monitoreo y control para el transporte público, orientado a mejorar la seguridad, la eficiencia y la experiencia de los usuarios.

Desarrollo

El proyecto integra sensores IoT, software de gestión en la nube y aplicaciones móviles para supervisar en tiempo real el flujo de pasajeros, condiciones del vehículo y rutas. Se utiliza inteligencia artificial para optimizar la asignación de recursos y tiempos de viaje.

Aplicabilidad

Puede implementarse en flotas de buses urbanos, interurbanos o transporte escolar, mejorando la puntualidad, reduciendo costos y aumentando la seguridad.

Escalabilidad

El sistema está diseñado para crecer desde una flota piloto de 5 vehículos hasta operaciones a nivel nacional sin cambios estructurales significativos.
